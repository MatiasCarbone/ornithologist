Ornithologist

Welcome to Ornithologist. This is an API used to classify audio recording of birds using deep learning.

The source code for this project is available on its GitHub page. 

Usage

Get Categories

The classification model has been trained on 40 species of birds from Argentina. To check the species 

the model can classify, make a request to /categories. This will return a list of all scientific names.

Make predictions

To make a prediction, send a POST request to /predict with an audio file in a valid  format.

The server will return a JSON file with multiple predictions, corresponding to slices of the audio file

where significant signal was detected. Each prediction will contain the scientific name of the bird, a 

probability score and a pair of timestamps (start-end), with the following structure:

{

      'label_code': int

      'species': str (scientific name)

      'probability': float (0-100)

      'time_section: (hh:mm:ss.mill, hh:mm:ss.mill)

  }
